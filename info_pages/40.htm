Intuit Certifications

Intuit QuickBooks: Help your students be career-ready

The Intuit QuickBooks Certified User certification is an industry-recognized credential that effectively validates one’s skills in the Intuit QuickBooks accounting software. Intuit QuickBooks Certified User exam vouchers will now include access to both the Intuit QuickBooks Desktop AND Intuit QuickBooks Online tests so candidates can demonstrate their skills in whatever version of Intuit QuickBooks they prefer. The Intuit QuickBooks software is free and can be downloaded here.

Exam versions

Intuit QuickBooks Certified User Online version

The Intuit QuickBooks Certified User Online exam will test candidates on new features found in the Intuit QuickBooks Online software. While many of the features are similar, Intuit QuickBooks Online is a cloud-based subscription service that provides the flexibility to take care of business anytime, anywhere. With the Online version, you are tested on the new features which include: receipt capture from a mobile device, cloud-based app support, work from anywhere on any device, instant file access for accountants, schedule and send invoices automatically.

Intuit QuickBooks Online version: Exam objectives | Exam languages and versions

Intuit QuickBooks Certified User Desktop version

The Intuit QuickBooks Certified User Desktop exam tests basic accounting knowledge and utilities of the Intuit QuickBooks Desktop software that includes pre-built sample companies, multiple company files, annual updates, and data backed-up and restored. Intuit QuickBooks Desktop is a one-time download accounting software with industry-specific features and reporting.

Intuit QuickBooks Desktop version: Exam objectives | Exam languages and versions

Intuit QuickBooks Online Proctoring

The newest format of the Intuit QuickBooks Certified User exams is Online Proctored (OP) exams. The Intuit QuickBooks online proctored exams allow students to conveniently and easily take an Intuit QuickBooks Certified User Online exam in the comfort of their home or office while being monitored by an offsite proctor.*

Learn more about Intuit QuickBooks online proctored exams

*All communication with the proctor is done in English
